# Hospital Project

A hospital project for the **Node & React** course using:

 - Node Js
 - React Js
 - Typescript

It includes the next modules:

 - Hospitals
 - Doctors
 - Authentication

## Running Node Project

    cd backend
    nvm use
    npm install
    npm run build:dev
    npm run debug

## Running React Project

    cd frontend
    nvm use
    yarn install
    yarn run start
